This is a simple Machine Learning project built for practical learning purposes.
The application predicts health insurance costs based on user input such as age, income, medical history, and lifestyle factors.

**📌 Project Purpose**
This project was created to:
Practice data preprocessing and feature engineering
Train and evaluate ML models
Export trained models using Joblib
Build a simple web app using Streamlit
Understand basic ML deployment workflow

⚠ This project is for learning and practice purposes only.
It is not intended for real-world insurance pricing use.
